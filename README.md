# WebWallpaper
## Introduction
A simple application that allows you to set a web page as a dynamic wallpape\
Note: The current version does not have a daemon process. Please make sure that the resource manager does not crash when using it, otherwise you need to restart the computer to restart WebWallpaper

Please download the latest version of Setup from [Releases](https://github.com/XiaoYuan151/WebWallpaper/releases) and you can use it after installation
